{% block main_title %}Update game{% endblock %}: pairs mode

Very important! FOLLOW THE SYNTAX! Write one pair of words every line, separated by a semicolon (;)

word1 ; word1_pair
word2 ; word2_pair
...

For example:

hi;hola
2+2;4
fruit ; apple
Brad Pitt; actor
Stephen Hawking ;scientific
